## What you will practice

- entering `VISUAL`, `VISUAL LINE`, and `VISUAL BLOCK`
- expanding or shrinking a selection with movement keys
- applying yank, delete, change, and paste to the current selection

<Aside type="note" title="Starting point">
	Use a file with several short lines so the effect of each selection mode is easy to see.
</Aside>

## Part 1: character-wise visual selection

<Steps>
	1. Start in normal mode and place the cursor at the beginning of a word.
	2. Press `v` to enter `VISUAL`.
	3. Press `l` or `h` to change the selection width character by character.
	4. Press `j` or `k` to extend the selection across lines.
	5. Press `y` to copy the selected text, or `Esc` to exit without changing the buffer.
</Steps>

## Part 2: line-wise selection

<Steps>
	1. Move to the first line you want to affect.
	2. Press `V` to enter `VISUAL LINE`.
	3. Press `j` to add the next line to the selection.
	4. Press `d` if you want to delete the selected lines, or `y` if you only want to copy them.
	5. Undo with `u` afterward if you are only practicing.
</Steps>

## Part 3: switch selection kinds in place

<Steps>
	1. Enter `VISUAL` with `v`.
	2. While the selection is active, press `V` to switch to `VISUAL LINE`.
	3. Press `Ctrl+v` to switch again to `VISUAL BLOCK`.
	4. Press `Esc` to leave visual mode entirely.
</Steps>

## Selection operations

<Tabs syncKey="visual-selection-ops">
	<TabItem label="Copy">
		Use `y` to yank the current selection into the slot.
	</TabItem>
	<TabItem label="Delete">
		Use `d` or `x` to delete the selection into the slot.
	</TabItem>
	<TabItem label="Change">
		Use `c` to delete the selection and enter insert mode immediately.
	</TabItem>
	<TabItem label="Replace">
		Use `p` to replace the current selection with the slot contents.
	</TabItem>
</Tabs>

## Practical drill

<Steps>
	1. Select one word with `v`.
	2. Yank it with `y`.
	3. Move somewhere else and paste with `p`.
	4. Enter `V` on a line and delete two lines with `d`.
	5. Undo with `u`.
</Steps>

## Important behaviors

- `F1` works inside visual mode and shows the visual-scope bindings.
- Movement keys still define the active edge of the selection.
- `Esc` leaves visual mode even if you have switched between multiple visual submodes.

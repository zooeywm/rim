## How to use this section

This section is task-oriented. Each tutorial is written for a first run:

- start from a concrete goal
- follow the exact keys to press
- learn the nearby navigation and recovery actions
- finish with the expected outcome and common detours

<Aside type="tip" title="Recommended order">
	Read these tutorials in sequence if you are new to `rim`: mode transitions, key hints, command palette, workspace file picker, notification center, and then the advanced pages for windows, tabs, and visual editing.
</Aside>

## Coverage

- These tutorials cover the built-in overlay modules and the primary mode-entry flow they depend on
- The pages assume default keymaps and default command aliases
- If you override `editor.toml`, `keymaps.toml`, or `commands.toml`, the UI structure stays the same but the exact keys may differ

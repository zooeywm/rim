## Release v0.0.3 <Badge text="Current focus" variant="tip" size="small" />

<Aside type="tip" title="Release theme">
	This release focuses on runtime consistency and configuration ergonomics.
</Aside>

<CardGrid>
	<Card title="Workspace session" icon="document">
		Workspace session persistence is aligned with runtime state, including tabs, windows, buffers, per-window view state, and scratch-buffer history.
	</Card>
	<Card title="Quit semantics" icon="warning">
		`:qa` is blocked when any buffer is dirty, and `:qa!`, `:wqa`, and `:wqa!` are implemented.
	</Card>
	<Card title="Command ergonomics" icon="setting">
		Command config supports inline table style for keymaps and command aliases.
	</Card>
	<Card title="Hot reload" icon="rocket">
		Config file changes are hot-reloaded through `FileWatcher` events with explicit full-table replacement semantics.
	</Card>
</CardGrid>

## Current feature set

<CardGrid>
	<Card title="Editing core" icon="pencil">
		Movement, insert, delete, paste, undo, and redo.
	</Card>
	<Card title="Visual modes" icon="document">
		`VISUAL`, `VISUAL LINE`, and `VISUAL BLOCK`.
	</Card>
	<Card title="Workspace model" icon="open-book">
		Multiple buffers, windows, tabs, and workspace session restore.
	</Card>
	<Card title="Runtime resilience" icon="warning">
		File watching, swap recovery, and persistent undo/redo.
	</Card>
	<Card title="Command line" icon="setting">
		Save, save as, reload, open file, and quit commands.
	</Card>
	<Card title="Portability" icon="rocket">
		Windows MSVC cross compilation via `cargo win-release`.
	</Card>
</CardGrid>

# Getting Started

## Prerequisites

- Rust toolchain with Cargo
- `rustfmt` nightly component for the repository formatting command
- `pnpm` if you want to run the docs site
- `yazi` if you want the external file picker flow

## Build And Run

```bash
cargo check -q
cargo run -p rim-app --
```

Open a file:

```bash
cargo run -p rim-app -- path/to/file.rs
```

## Workspace Orientation

- app entry point: `rim-app/src/main.rs`
- runtime shell: `rim-app/src/app.rs`
- application dispatch: `rim-application/src/action_handler/`
- pure editor logic: `rim-domain/src/editor/`
- adapters: `rim-infra-*`

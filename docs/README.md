# rim Docs

This directory hosts the Astro Starlight documentation site for `rim`.

## Local Development

```bash
pnpm install
pnpm dev
```

## Production Build

```bash
pnpm build
pnpm preview
```

## Content Layout

- `src/content/docs/guides/`: onboarding and operational documentation
- `src/content/docs/reference/`: feature, architecture, and command reference
- `astro.config.mjs`: Starlight navigation and site metadata
